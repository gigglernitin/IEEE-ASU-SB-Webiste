IEEE Career

Date: 29/4/2017

Location: Faculty of engineering, ASU

  • Description:

    • What is IEEE Career?!

      IEEE is opening the opportunity to enhance and strengthen your technical skills through affording variety of technical packages and orientation sessions concerning the real world jobs with zero fees… Vodafone is contributing with us to bring you opportunities that can never be missed.

      FIRST PACKAGE: (Mobile Package by Vodafone)

      -For all undergraduate students interested in Mobile Communication field, a full Mobile Package will be given.

      -Contents: GSM, 1G, 2G, 3G and 4G

      SECOND PACKAGE: (Embedded System Package)

      -A full Diploma provided by Eng. Ahmed Abd El-Ghafar, an engineer in The Engineering Authority of Military Forces.

      -Contents: C -> C Embedded, Data Structures, Software Engineering, Electronics and Logic Components, Interfacing(PIC & ARM). RTOS(Real Time Operating System).

      -A Certificate signed by the Engineers Syndicate will be delivered to the participants.

      THIRD PACKAGE: (4IPB Company Packages)

      -4IPB is an IT and Big Data Company occupied in Alexandria.

      -This package will be divided into 3 different tracks concerning about big important majors.

      1 - Moving to Cloud - Docker for Developer:

      What is Docker? What problems does Docker solve? Why is Docker important? Where and when to use Docker Example: “Hello, World” Docker command line Controlling containers: building a website monitor Solved problems and the PID namespace Building environment-agnostic systems Building durable containers What is a repository? Docker Hub from the command line Installation files and isolation

      2 - Intro to VMware Vsphere 6.x :

      Introducing VMware vSphere 6.x Planning and Installing VMware ESXi Installing and Configuring vCenter Server vSphere Update Manager and the vCenter Support Tools Creating and Configuring Virtual Networks Creating and Configuring Storage Devices Creating and Managing Virtual Machines Using Templates and vApps

      3 - Intro to Red Hat System Administration:

      Intro to linux Software package management User , Groups Disk Partitioning Filesystem , Swap Boot and Run control levels Backup, Restore and compression job scheduling and system log Intro to network, network interface and routing DNS and DHCP NIS and LDAP Internet Service and E-Mail Active Directory and Samba System Security

      -Each participant will be able to enroll in only one track.

      Fourth Package: (IEEE Orientation Session)

      -This session will be provided by different IEEE chapters, in different universities.

      -Chapters as RAS and PES.

      -The session will aim to spread knowledge towards these chapters before establishing their Branches in ASU during 2018.
